- **5GC**: 5G Core  
- **5GS**: 5G System  
- **AAL**: Accelerator Abstraction Layer

- **NF**: Network Function

- **NG-RAN**: Next Generation RAN  
- **NGAP**: Next Generation Application Protocol  
- **NIST**: National Institute of Standards and Technology  
- **NMS**: Network Management System
